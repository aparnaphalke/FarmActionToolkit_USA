# FarmActionToolkit_USA (FAcT-USA)

Decision-support oriented Earth Observation (EO) workflows for U.S. agriculture, enabling scalable monitoring of cropland extent, crop type, and planting dates.

---

## Overview

The Farm Action Toolkit for the United States (FAcT-USA) is an Earth Observation-based framework designed to generate spatially explicit agricultural information to support monitoring, modeling, and decision-making. The toolkit integrates multi-source satellite data and phenology-guided approaches to produce:

- Cropland extent maps  
- Crop type classification  
- Planting date estimation (Day of Year, DOY)  

These products support applications such as crop modeling, drought monitoring, irrigation planning, and agricultural risk assessment.

---

## Dataset

Public dataset (DOI):  
https://doi.org/10.5281/zenodo.19671983  

The dataset includes:

- Pixel-level (30 m) rasters  
- Census block group-level aggregates  
- County-level aggregates  

All datasets are derived from satellite observations and are designed to be reproducible and scalable.

---

## Alabama Implementation (2024)

The workflow is implemented independently for each Agricultural Statistics District (ASD):

- ASD 110  
- ASD 120  
- ASD 130  
- ASD 140  
- ASD 150  
- ASD 160  

For each ASD:

- Zone-specific cropland masks and crop type maps are used  
- Planting dates are estimated separately  
- Final statewide products are generated by combining outputs across all ASD zones  

Planting dates for Alabama corn are constrained within an agronomically plausible range:

- DOY 88–162 (March–June)

---

## Key Features

- Fully EO-driven approach  
- Phenology-guided planting date estimation  
- No reliance on assumed or temperature-based planting dates  
- Multi-scale outputs (pixel, block group, county)  
- Scalable to other regions and crop types  
- Designed for integration with crop and hydrologic models  

---

## Reproducibility

All workflows are implemented in Google Earth Engine (GEE) and are designed to be transparent, reproducible, and extensible.
